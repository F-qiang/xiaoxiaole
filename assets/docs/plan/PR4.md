# PR4（特效、障碍物与结算逻辑）

## 目标
完成特效棋子生成、触发、连锁扩散、障碍物清除与胜负判定，形成完整规则闭环。

## 任务 13：实现特效棋子生成
- **目标**：在特定消除组合后生成特效棋子。
- **实现内容**：
  - 竖向 4 连生成行清除特效，横向 4 连生成列清除特效
  - 直线 5 连生成螺旋炸弹 / Color Bomb，触发后清除全棋盘同色
  - 特效生成在组合中心格
  - 保留原棋子的颜色属性，遵循同色消除规则
  - 2×2 生成范围爆炸特效
  - T 型生成炸药桶特效
- **验收方式**：
  - 特定消除后能出现对应特效棋子
  - 特效生成位置正确
  - 颜色属性与消除规则一致
  - 螺旋炸弹触发后能正确清除全棋盘同色

## 任务 14：实现特效触发
- **目标**：特效棋子能按规则触发。
- **实现内容**：
  - 范围爆炸自动触发
  - 行清除自动触发
  - 螺旋炸弹可与任意特效或棋子交换触发
  - 炸药桶手动交换触发
  - 炸药桶交换后无论是否形成普通消除，均直接触发爆炸
  - 炸药桶爆炸可触发范围内其他自动类特效的连锁反应
- **验收方式**：
  - 特效触发后效果正确
  - 炸药桶不会被普通消除自动引爆
  - 炸药桶可引发进一步连锁
  - 螺旋炸弹可正确触发全棋盘同色清除

## 任务 15：实现特效连锁扩散与组合技
- **目标**：特效之间可形成连锁反应，并支持标准组合技。
- **实现内容**：
  - 队列式广度优先扩散
  - 连锁范围内自动特效继续触发
  - 火箭 + 炸弹：同时触发行清除与范围爆炸，按两者作用范围合并处理
  - 炸弹 + 炸弹：扩大范围清除效果，按更大范围执行
  - 螺旋炸弹 + 任意特效：触发全棋盘同色清除，并叠加目标特效效果
- **验收方式**：
  - 多个特效可连续引爆
  - 连锁流程符合规则文档
  - 组合技触发结果正确

## 任务 16：实现障碍物清除
- **目标**：障碍物可被特效或相邻消除清除。
- **实现内容**：
  - 特效范围覆盖障碍物时清除
  - 相邻消除采用四邻域判定（上下左右），不含斜角
  - 同一障碍物被多个消除 / 特效同时覆盖时，仅清除 1 次，计数只减 1
  - 障碍物剩余数量同步减少
- **验收方式**：
  - 障碍物清除正确
  - 顶部数量实时更新
  - 重复覆盖不会重复计数

## 任务 17：实现胜负判定
- **目标**：正确判断胜利和失败。
- **实现内容**：
  - 障碍物为 0 时胜利
  - 盘面稳定且无可用交换时失败
  - 结算弹窗展示结果
- **验收方式**：
  - 胜利和失败都能正确触发
  - 结算时机符合设计文档

## PR4 联调验证
- **边界场景测试**：覆盖角落特效、多特效连锁、障碍物重复清除、操作锁定等边界 Case。
- **特效规则验证**：检查 4 连生成、5 连螺旋炸弹、T 型炸药桶、炸药桶触发和连锁扩散是否与规则文档一致。
- **障碍物清除验证**：重复覆盖、四邻域判定和剩余数量扣减是否正确。
- **组合技验证**：火箭 + 炸弹、炸弹 + 炸弹、螺旋 + 任意的触发结果是否符合预期。
